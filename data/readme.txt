cluster_cen.txt
the position of cluster/AOI centers, each row contains the position X,Y of a cluster center

fix_data.txt
the position and duration of fixations, each row contains a scanpath composed of fix_x, fix_y and duration

fix_label.txt
the cluster/AOI label of each fixation, there is a one-to-one correspondence between the label and fixations in fix_data.txt
